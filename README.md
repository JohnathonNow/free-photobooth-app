# free-photobooth-app
A free photo booth website. Live [here](https://johnathonnow.github.io/free-photobooth-app/), it can also be downloaded and used offline.

It currently offers a variety of countdown options, as well as options for frames, overlays, filters, and more. The settings menu can optionally be hidden to operate as more of a kiosk than as a customizable app. Settings are stored in the URL, allowing you to bookmark specific settings or set up multiple kiosks very quickly.

It can be combined with kiosk mode and kiosk printing flags for a more seamless experience. For details on that, see [this guide](https://gist.github.com/sajinct/cf4863f7b5da061b2c65c104c55a6da6).
